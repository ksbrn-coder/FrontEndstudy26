HTML 학습 가이드

HTML에서 제목을 만들 때는 <h1> 태그를 사용합니다.

기본 HTML 구조

<!DOCTYPE html> 
<html lang="ko"> 
<head>
 <meta charset="UTF-8">
 <title>제목</title>
</head>
<body>
 <h1>안녕하세요!</h1>
</body>
</html>
